# Security Policy

## Supported Versions

Security fixes are considered for the latest public release.

## Reporting A Vulnerability

Please report security issues privately by email or LinkedIn rather than opening
a public issue.

Include:

- the affected script, crawler, or generated file path;
- a short reproduction or explanation;
- whether personal data, cookies, credentials, or application history could be
  exposed;
- any suggested mitigation.

## Scope

This project can process local job-search profiles and generated application
trackers. Treat profile configuration, workbook outputs, crawler caches,
browser/session data, and diagnostics as privacy-sensitive.

Do not commit local profiles, generated workbooks, private job notes, credentials,
cookies, `.env` files, or exports containing personal application history.
